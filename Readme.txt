Project name: Movie Trailer Website
Course: Machine Learning Foundation Nanodegree Program

Movie trailer website project is a project made using Flask web development framework in Python.

Depedencies:
from flask - Flask, render_template
requests

Working:
Project requests movie data from https://www.themoviedb.org
Movie data comes under four categories, namely - TOP RATED, POPULAR, NOW PLAYING and UPCOMING
Each section has list of movies, upon clicking trailer page is opened
